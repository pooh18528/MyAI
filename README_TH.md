# MyAI CLI

**CLI** ย่อมาจาก **Command Line Interface** (อินเทอร์เฟซแบบบรรทัดคำสั่ง) คือโปรแกรมที่เราสั่งงานด้วยการ "พิมพ์คำสั่ง" แทนการใช้เมาส์คลิกปุ่มบนหน้าจอ

โปรแกรมนี้ช่วยให้คุณพูดคุยกับ Gemini AI และสั่งให้ AI ช่วยแก้โค้ดในไฟล์ของคุณได้ทันทีผ่านหน้าจอดำๆ (Terminal)


## การใช้งาน

เริ่มโปรแกรม:
```bash
myai
```

### คำสั่งที่มี:
- **คุยเล่น**: พิมพ์คุยกับ AI ได้เลยเหมือนแชทปกติ
- **/fix file.js**: สั่งให้ AI แก้บั๊กในไฟล์ (ระบุชื่อไฟล์)
- **/reset**: ลบ API Key (เพื่อใส่ใหม่)
- **exit**: ปิดโปรแกรม

## การตั้งค่า
เมื่อเข้าโปรแกรมครั้งแรก ระบบจะถามหา **API Key** อัตโนมัติ
สามารถไปกดรับฟรีได้ที่: [Google AI Studio](https://aistudio.google.com/app/apikey)
